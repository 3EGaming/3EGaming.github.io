Getting Over It Multiplayer Mod Install Guide

About

This is the install guide for the Getting Over It Multiplayer Mod. For information about the mod please visit this page.

Setting up BepInEx

First we need to install a tool called BepInEx which will let us load the multiplayer code into the game. To do this we need to download the latest release x64 version which can be found on the Releases page.

Next go to Getting over it in your steam library, go to the cog, manage, browse local files.

You should now see a folder with several files.

Open up the file we downloaded from BepInEx and copy its contents to the Getting over it folder.

You should now have these files in the game's folder:

Open the BepInEx folder and create a new folder inside called "plugins".

You now have BepInEx installed to your game files, but we're not done yet. Keep the plugins folder open and move to the next step.

Setting up the Mod

Next we need to download the mod and add it to the plugins folder.
Go to the Downloads page and download the latest version.
Open the file and copy the .dll file into the plugins folder.

The mod is now fully installed!

Using the Mod

Now that the mod is installed, here's how to use it. Launch the game through steam and you should see a message at the top of the screen to show that the mod is working correctly. You can press the space key to dismiss this message.

Either continue or start new game and open the pause menu.

Select whether you want to be host or join an exisitng game. There will be 1 host and other players will join that host's code.

Host

To host, click host.

You can now click copy to add the code to your clipboard so you and paste it in a chat to share with other players.

Join

To join, click join.

Now copy the code that the host sent you and click the box above the join button and press CTRL+V to paste the code, and finally press the join button.

Now you are in a lobby with your friends and you can climb together. Have fun!

Updates

To update the game simply delete the .dll file from the plugins folder and redo the steps from Setting up the mod, but download the newer version on the Downloads page.
