Getting Over It Multiplayer Mod

About

Getting Over It Multiplayer Mod is a mod which allows you to play Getting Over It with up to 250 climbers.

Credits

Thanks of course to Bennett Foddy for making such an interesting and fun game. I would definitely recommend it to anyone who doesn't already own it.
Thanks to ScottGameDev for testing, feedback, and suggestions.

Please read before downloading

Although I have tested the mod and checked the code over several times to make sure the code works correctly, you use this mod at your own risk and I am not responsible for anything that happens.
You should also know that the mod is only available for the Steam version of the game on windows, which you will need to own to use the mod. You can buy it here.
