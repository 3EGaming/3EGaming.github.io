GrappleVR Mod

About

GrappleVR is a mod for Grapple which allows you to play in VR.

Please read before downloading

Although I have tested the mod to make sure it works correctly, you use this mod at your own risk and I am not responsible for anything that happens.
Please keep in mind that this is probably not allowed to be used in speedruns.

Installation

First, download the latest version of BepInEx, and the latest version of the mod.

Next, open the game's files from the steam library:

Next, extract the files from both downloaded zip files to the game's files.
Once you have done it should look like this:

Now you just need to connect your VR headset and make sure it is set as the active OpenXR runtime, and then launch the game through steam.
