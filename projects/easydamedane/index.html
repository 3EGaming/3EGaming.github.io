Troubleshooting:

Can't connect to gpu

This is likely because Google colab only lets you use their GPUs for a certain amount of time and sometimes other people are already using them.

The only solution is to wait for a GPU to become avalible (try again later) or buy Google colab pro.

Can't upload file/file upload button not showing

This could be because your browser isn't supported for uploading to colab

Try using Google chrome

Nothing is happening after upload

The script takes a few minutes to process so please wait a few minutes

If it takes more than 10 minutes try reloading the page as you may have acidentaly pressed backspace on some of the code

Geting errors

Try reloading the page as you may have acidentaly pressed backspace on some of the code

There also may be an error on colab or my scripts so please try again later

*output may not be perfect so trimming to around 12-13 seconds is recomended
